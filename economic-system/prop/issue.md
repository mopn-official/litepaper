| ATT Gem       | 2,000            |
| STA Gem       | 2,000            |
| Venom         | 10               |
| Disarm Field  | 10               |
| Iced Cage     | 5                |
| Pause Capsule | 5                |
| Oscillator    | 5                |
| Purge Laser   | 5                |
| Terminator    | 1/500            |

Terminator：1 Terminator will be issued every 500 days
